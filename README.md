# Income Expense Calculator

A fully functional web application for tracking income and expenses with full CRUD operations and local storage persistence.

# Features

- **Add Transactions**: Create new income or expense entries with description and amount
- **Edit Transactions**: Modify existing entries with ease
- **Delete Transactions**: Remove unwanted transactions with confirmation
- **Filter Transactions**: View all transactions, or filter by income or expense only

# Metrics

The dashboard displays three key metrics:
- Total Income (green)
- Total Expenses (red)
- Net Balance (blue/green for positive, red for negative)
